# 🤝 Contributing to Free AI Engineer Course

🎉 Thank you for your interest in contributing!

---

## 📜 Guidelines

### ✅ What We Accept

- 📝 **New Lessons** - Add more days to existing weeks
- 🐛 **Bug Fixes** - Fix errors in existing content
- 📖 **Better Explanations** - Improve clarity
- 🌐 **Translations** - Translate to other languages (Hindi, Spanish, etc.)
- 🖼️ **Visuals** - Add diagrams, flowcharts, roadmap images
- 📚 **Resources** - Add useful links and references

---

## 🚀 How to Contribute

### 1. Fork the Repository

Click the **Fork** button on GitHub

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/free-ai-course.git
cd free-ai-course
```

### 3. Create a Branch

```bash
git checkout -b add-new-lesson
```

### 4. Make Changes

- Add lessons to appropriate Week folder
- Update README.md if adding new content
- Follow existing naming convention: `DayX.md`

### 5. Commit & Push

```bash
git add .
git commit -m "Add: New lesson on [topic]"
git push origin add-new-lesson
```

### 6. Create Pull Request

Open GitHub and create a **Pull Request** to the main repo
